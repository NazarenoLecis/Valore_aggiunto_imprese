# Dizionario dati

country_code indica il codice del paese.

country_name indica il nome del paese.

year indica l’anno dell’osservazione.

source_name indica la fonte.

source_dataset_id indica il dataset di origine.

sector_code_original indica il codice settoriale nella fonte.

sector_code_harmonised indica il codice settoriale armonizzato.

size_class_original indica la classe dimensionale nella fonte.

size_class_harmonised indica la classe dimensionale armonizzata.

metric_code indica la variabile misurata.

value indica il valore osservato o stimato.

unit indica l’unità di misura.

method_status indica lo stato metodologico del dato.

quality_flag contiene eventuali flag di qualità della fonte.

download_timestamp indica quando il dato è stato scaricato.
