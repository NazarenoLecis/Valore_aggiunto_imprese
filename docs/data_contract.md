# Contratto dati

I dataset finali sono salvati in CSV e JSON.

La chiave logica minima è composta da paese, anno, fonte, settore, classe dimensionale e metrica.

Ogni dataset deve includere method_status per distinguere dati osservati, dati di distribuzione, stime e dati non disponibili.

I file JSON usano orientamento a record. Ogni riga del CSV corrisponde a un oggetto nel JSON.
